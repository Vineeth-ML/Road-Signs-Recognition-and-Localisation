# Road-Signs-Recognition-and-Localisation

Road Sign Detection & Classification
A dual-purpose deep learning model that classifies road signs and predicts their bounding boxes in a single forward pass.
Built using a ResNet18 backbone with two output heads one for classification, one for localization.
Trained and validated on a road sign dataset. Both loss curves decline sharply early on, then plateau indicating strong convergence and good generalization to unseen data.
